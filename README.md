# 🎵 MusicMate – A Social Media Music Web Application

MusicMate is a full-stack social music-sharing platform that blends the 
best of Spotify and Instagram. Built as an MCA final year project, it 
lets users stream music, share stories, manage playlists, and connect 
with others through real-time chat — all in one place.

## ✨ Features
- 🔐 **User Authentication** – Signup, login, email verification & profile setup
- 🎧 **Music Streaming** – Browse, search, and play songs with an embedded player
- ❤️ **Favourites** – Like and save songs to a personal library
- 📖 **Story Sharing** – Upload and view stories; click a story to open a chat
- 💬 **Real-Time Chat** – Direct messaging between mutual followers
- 🎼 **Playlist Management** – Create, edit, rename, delete & share playlists
- 👤 **Profile Management** – Edit profile, follow users, view follower updates

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Tools:** VS Code, Git, XAMPP

## ⚙️ How to Run
1. Clone the repository
2. Import `database/musicmate.sql` into MySQL
3. Configure your DB credentials in `config.php`
4. Run on a local server using XAMPP or WAMP
5. Open `index.php` in your browser
